Project Description: Student Academic Management System (SAMS)
SAMS is a web‑based platform that centralizes all academic services for students and administrators. Built with PHP, MySQL, HTML, CSS, and Bootstrap, it provides a complete solution for managing student information, course registration, results, attendance, continuous assessment, teacher evaluation, exam clearance, and various student applications (certificate, transcript, scholarship, laptop, transport card, convocation).

Key Features:

Student Panel: Dashboard, profile, registered courses, results & CGPA, continuous assessment marks, attendance, advisor info, teacher evaluation, application submission, exam clearance, and password change.

Admin Panel (Role‑based): Three admin levels (Senior DBA, Junior DBA, DBA Trainee) with granular permissions. Manage students, teachers, courses, applications, generate reports, view activity logs.

Authentication: Secure login/registration with password hashing (SHA‑256) and session management.

Benefits:

One‑stop access to academic information

Reduces manual workload

Real‑time data updates

Audit trail for admin actions

Scalable and easy to deploy on XAMPP or any PHP/MySQL server.

SAMS is ready for use in universities and educational institutions to streamline academic management.
